## ✨ Fitur Utama

### 📋 Manajemen Siswa
- ✅ Tambah, edit, dan hapus data siswa
- 🏷️ Status siswa (Aktif / Lunas / Tugas Terlambat)
- 📚 Pilih mata pelajaran (Matematika, IPA, B. Inggris, Calistung)
- 📊 Export data siswa ke Excel (`.xlsx`)

### 💰 Manajemen Keuangan
- 💵 Catat pembayaran SPP per siswa
- 📅 Pilih bulan & tahun pembayaran
- 📈 Total pemasukan otomatis
- 🗑️ Hapus transaksi yang salah

### ✅ Sistem Absensi
- 📝 Absensi harian (Hadir / Izin / Sakit / Alpa)
- 📅 Filter absensi berdasarkan tanggal
- 📊 Export data absensi ke Excel
- 🎯 Statistik kehadiran hari ini

### 🔐 Keamanan & UX
- 🔑 Login admin dengan session
- ☁️ Data tersimpan di **Firebase Realtime Database**
- 📱 Responsive design (mobile-friendly)
- 🎨 UI modern dengan animasi smooth
- 🔔 Toast notification untuk feedback
- ⌨️ Keyboard shortcut (Enter untuk submit)
